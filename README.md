# AxCore
>Minecraft plugin that adds all essential commands to a minecraft server
---
## Features
- Day
- Feed
- Fly
- FlySpeed
- GameMode
- Heal
- MOTD
- Night
- Ping
- Weather
---
## Installation
1. Download the latest release
2. Put the `.jar` into the `/plugins` folder of every server to be synchronized
3. Start the server
