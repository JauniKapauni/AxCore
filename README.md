# AxCore
>Minecraft plugin that adds all essential commands to a minecraft server
---
## Requirements
- Minecraft Paper 26.1.2 Server
---
## Features
- Day
- Feed
- Fly
- FlySpeed
- GameMode
- Heal
- MOTD
- Night
- Ping
- Weather
---
## Installation
1. Download the latest release
2. Put the `.jar` into the `/plugins` folder
3. Start the server
---
## Commands & Permissions
- `/day` - `axcore.day`
- `/feed` - `axcore.feed`
- `/fly` - `axcore.fly`
- `/flyspeed` - `axcore.flyspeed`
- `/gamemode` - `axcore.gamemode`
- `/heal` - `axcore.heal`
- `/motd` - `axcore.motd`
- `/night` - `axcore.night`
- `/ping` - `axcore.ping`
- `/weather` - `axcore.weather`
